# PlayerStats

A comprehensive character statistics addon for WoW TBC Classic. Provides a stats panel and kill tracking system, bringing functionality similar to the character stats pane introduced in Wrath of the Lich King to TBC.

## To Install

If installing via zip, extract to your WoW Anniversary folder. Otherwise just install via Curseforge.

## Features

- Full character stats panel with a toggleable main window and compact mini bar mode
- Kill tracking with per-session and per-character lifetime statistics
- Killstreak tracking with personal best records
- Killstreak announcements to chat (say, emote, party/raid, or all channels)
- Session stats that persist through logout and can be reset on demand
- Per-character saved data

## Slash Commands

- `/ps` - Toggle the stats window
- `/ps mini` - Toggle the compact mini bar
- `/ps session` - View current session stats
- `/ps session reset` - Reset the current session
- `/ps streak` - View your best kill streak
- `/ps reset` - Reset current character stats
- `/ps announce <none|say|emote|group|all>` - Set killstreak chat announcements
